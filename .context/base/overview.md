## Project overview

Photobon Landing is a static marketing site for the Photobon brand, a portable mini photobooth product with receipt-style photo output for events, cafes, tenants, and brand activations.

### Primary user

- Potential venue, event, and brand customers evaluating Photobon and starting contact via WhatsApp.

### Current value delivered

- Presents the Photobon brand and offer in Indonesian.
- Explains use cases, placement examples, and contact/discussion prompts through static content files.
- Routes users toward WhatsApp-based conversion rather than an internal backend flow.

### Current scope

- File-based Astro routes for `/`, `/cara-kerja`, `/penempatan`, and `/kontak`.
- A fully composed home page.
- Scaffolded secondary routes with typed content already prepared in `src/content/`.

### Explicit non-goals from repository evidence

- No CMS-backed content editing.
- No authenticated user area.
- No API/server-side data layer.
- No payment, booking, or submission persistence inside the app.
